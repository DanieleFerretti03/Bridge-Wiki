# Per ottenere il pdf:
Il progetto e' in latex con compilatore lualatex.
Potete compilare a mano il progetto oppure usare il comando dentro il file make.

Se volete compilare il progetto usate il comando (avendo le dipendenze installate):
```bash
make
```
